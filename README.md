## General info
Full Stack Web Application with serverless backend
	
## Technologies
Project is created with:

Client:
* HTML5
* CSS3
* Node.js
* React Router Dom v6.2.2
* Bootstrap v5.1.3
* Material UI v0.0.1
* Fuse v0.4.0
* Fuse.js v17.0.2

Server:
* cors v2.8.5	
* dotenv v16.0.0
* express v4.17.3
* mongo db v4.4.1


## Setup

Client: 
 * npm install relevant packages
 * npm start

Server:
* npm install relevant packages
* Create PORT and MONGO_URI Variable and specify Value.
* npm start
